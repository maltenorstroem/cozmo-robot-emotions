# Robot emotions
inspired by ANKI cozmo

## Available emotions
- standard mood
- smiling
- giggling
- check right
- check left
- giggling
- dubiative
- sad
- really sad
- bored
- kill all the humans

## How to use
just call the desired function on the robot-emotions component.
e.g. in dev console $0.smile();

## Install dependencies
> npm install
## Run the sample
> npm run start
